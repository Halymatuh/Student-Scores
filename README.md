# Student Scores CSV Manager

Overview

This is a Python program that stores student names and scores in a CSV file. After saving the records, it reads the file, calculates the class average, and identifies the student with the highest score.


 Features

- Add student names and scores
- Save records to a CSV file
- Read data from a CSV file
- Calculate the class average
- Find the highest-scoring student
- Display results to the user

 How to Run

1. Open a terminal in the project folder.
2. Run:

```bash
python student_scores.py
```

3. Enter student names and scores when prompted.
4. View the class average and top performer after data entry is complete.
